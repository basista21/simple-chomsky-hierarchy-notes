Type 0 Grammar

Type 0 grammars generate recursively enumerable languages, which is the type of language accepted by a Turing machine. This type of grammar includes all formal grammars.

Example 1

All regular, context-free, and context-sensitive languages are recursively enumerable. An example of a recursively enumerable language that is not at the same time context-sensitive is the set of Turing machines that halt.

Type 1 Grammar

Type 1 grammars generate context-sensitive languages, which is the type of language accepted by a linear bounded automaton. In this type of grammar, given a set of production rules of the form LHS → RHS, the LHS should be of the form αΑβ → αγβ, where Α is a nonterminal, while α, γ, and β can be terminals or nonterminals and γ is not an empty string.

Example 1

The grammar defined by the set of nonterminal symbols {A, B, S}, terminal symbols {a, b, c}, start symbol {S}, and production rules
  S → aSAB
  S → aAb
  BA → AB
  CB → cc
  aA → ab
  bA → bb
  bB → bc
is a Type 1 grammar.

This generates the context-sensitive language anbncn where n is a positive integer. This is an example of a Type 1 grammar that is not at the same time Type 2.

Type 2 Grammar

Type 2 grammars generate context-free languages, which is the type of language accepted by a nondeterministic pushdown automaton. In this type of grammar, given a set of production rules of the form LHS → RHS, the LHS should be a single nonterminal symbol, while the RHS can be a string of nonterminals and terminals.

Example 1

The grammar defined by the set of nonterminal symbols {A}, terminal symbols {a, b}, start symbol {A}, and production rules
  A → ab
  A → aAb
is a Type 2 grammar.

This generates the context-free language anbn where n is a positive integer. This is an example of a Type 2 grammar that is not at the same time Type 3.

Type 3 Grammar

The most restrictive type of grammar and is pictured in the diagram above as the innermost circle. Type 3 grammars generate regular languages, which is the type of language accepted by a finite-state automaton. In this type of grammar, given a set of production rules of the form LHS → RHS, the LHS should be a single nonterminal symbol, while the RHS can only be either a single terminal symbol or one terminal symbol with one nonterminal symbol.

Example 1

The grammar defined by the set of nonterminal symbols {A}, terminal symbols {a}, start symbol {A}, and production rules
  A → a
  A → Ʌ
is a Type 3 grammar.

Example 2

The grammar defined by the set of nonterminal symbols {A}, terminal symbols {a}, start symbol {A}, and production rules
  A → aA
  A → a
is a Type 3 grammar.

This generates the regular language an where n is a positive integer.

Formal Grammar

Informal Definition

A grammar consists of a set of production rules (also called rewriting rules), usually written in the notation LFS → RHS. Both sides are composed of symbols in the alphabet of the grammar. These symbols can be classified into nonterminal symbols (usually represented by uppercase letters), terminal symbols (usually represented by lowercase letters), and a start symbol.

Example 1

The set of nonterminal symbols {A}, terminal symbols {a}, start symbol {A}, and production rules
  A → a
  A → aA
define a grammar.

This example grammar generates a language of only the symbol 'a', reminiscent of a certain programming language.

Quote

“Language is a process of free creation; its laws and principles are fixed, but the manner in which the principles of generation are used is free and infinitely varied.

Noam Chomsky in For Reasons of State
